# ⚙️ CPU Process Scheduler

A web-based CPU scheduling simulator built with pure HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 [Click here to open]
(https://vaishakhpuriya-alt.github.io/cpu-scheduler)

## 📌 Features
- Supports 6 scheduling algorithms
- Interactive Gantt chart visualization
- Auto-calculates Waiting Time, Turnaround Time & CPU Utilization
- Add / remove processes dynamically
- Clean dark-themed UI

## 🧠 Algorithms Supported
| Algorithm | Type |
|-----------|------|
| FCFS | First Come First Serve |
| SJF | Shortest Job First |
| SRTF | Shortest Remaining Time First |
| Round Robin | With custom Time Quantum |
| Priority | Preemptive (higher number = higher priority) |
| HRRN | Highest Response Ratio Next |

## 🛠️ How to Use
1. Select a scheduling algorithm
2. Add processes with Arrival Time and Burst Time
3. Click **Run Scheduler**
4. View the Gantt Chart and results instantly

## 📂 Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
